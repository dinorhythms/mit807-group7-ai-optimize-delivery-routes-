# MetroSwift Web App - Quick Start

## Run Locally in 2 Steps

### 1. Install Flask & Gunicorn

```bash
pip install Flask gunicorn
```

### 2. Start the Web Server

```bash
python web_app.py
```

## Access the Web App

Open your browser and go to:

```
http://localhost:5000
```

## What You'll See

- Modern web interface with the same algorithms (BFS, DFS, A\*)
- Interactive controls for route optimization parameters
- Real-time results display
- Professional, responsive design

## Features

✅ Select source and destination locations
✅ Adjust traffic, peak, weather, and reroute parameters
✅ Run three different search algorithms
✅ View traversal order and optimal paths
✅ See metrics (distance or travel time)
